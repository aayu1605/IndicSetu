# IndicSetu
# 🌾 Indic-Setu - Government Schemes Made Simple

## 🎯 Mission
Help 500M+ rural Indians discover government schemes they're eligible for

## ✨ Features
- 🌐 10+ Languages (Hindi, Gujarati, Marathi, Tamil, Telugu, Kannada, Bengali, Punjabi, Urdu, English)
- 🎤 Voice Input & Output
- ❤️ Save Favorite Schemes
- 📜 Search History
- 📊 Smart Analytics
- 📱 Mobile Optimized
- 🎨 Beautiful UI with Custom Bot Mascot
- 🆓 Free Forever
- ⚡ Lightning Fast

## 📋 Supported Schemes
1. PM-Kisan (₹6,000/year)
2. MGNREGA (100 days work/year)
3. PMJDY (Free bank account)
4. Ayushman Bharat (₹5 lakh health)
5. PMSBY (₹2 lakh insurance)
6. PM-Mandhan (₹3,000/month pension)
7. Kisan Credit Card (Agri loans)
8. Pradhan Mantri Awas Yojana (Housing)
9. Sukanya Samriddhi (Girl savings)

## 🚀 How to Use
1. Visit: [Your Streamlit URL]
